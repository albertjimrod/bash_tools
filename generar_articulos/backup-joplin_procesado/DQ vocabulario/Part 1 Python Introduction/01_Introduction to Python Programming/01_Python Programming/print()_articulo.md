# Descubriendo el Mundo de los Números con Print Statements: Un Viaje desde lo Sencillo hasta lo Complejo

## Comenzando con lo Simple: Sumar Números Decimales

Consideremos el primer ejemplo:

```python
print(0.00 + 1.99)
```

Cuando ejecutamos este código, vemos que se imprime `1.99`. Aquí, estamos sumando dos números decimales (también conocidos como flotantes). El primero es 0.00 y el segundo es 1.99. Al sumarlos, obtenemos 1.99.

Esto nos muestra una de las muchas cosas que podemos hacer con los números en Python: operaciones básicas. Los números decimales son especialmente útiles cuando necesitamos realizar cálculos precisos, como en finanzas o en cualquier situación donde los centavos importan.

## Llevando las Cosas al Siguiente Nivel: Restar Grandes Números

