# Introducción a las estructuras condicionales en Python: El ejemplo de `if` statements

Python es un lenguaje de programación versátil y fácil de aprender. Una de sus características más utilizadas son las estructuras condicionales, que permiten al código tomar decisiones basadas en ciertas condiciones.

En este artículo, exploraremos el uso de `if`, `elif`, y `else` mediante un ejemplo sencillo que pedirá al usuario ingresar un número entero y mostrará diferentes mensajes dependiendo del valor ingresado.

## El código base

Aquí está el código original:

```python
x = int(input("Please enter an integer: "))

if x < 0:
    x = 0
    print('Negative changed to zero')
elif x == 0:
    print('Zero')
elif x == 1:
    print('Single')
else:
    print('More')
```

## Comprendiendo el código paso a paso

1. **Obteniendo la entrada del usuario:**
   ```python
   x = int(input("Please enter an integer: "))
   ```
   - La función `input()` se utiliza para leer una entrada del teclado.
   - La conversión a entero se realiza utilizando `int()`.
   
2. **Estructura condicional con `if`, `elif`, y `else`:**
   
   - **Condición 1: Si el número es menor que cero (`x < 0`):**
     ```python
     if x < 0:
         x = 0
         print('Negative changed to zero')
     ```
     - Si el usuario ingresa un número negativo, se cambia `x` a cero.
     - Se imprime un mensaje informando que el número negativo fue cambiado a cero.
   
   - **Condición 2: Si el número es igual a cero (`x == 0`):**
     ```python
     elif x == 0:
         print('Zero')
     ```
     - Si el usuario ingresa cero, se imprime un mensaje informando que el número es cero.
   
   - **Condición 3: Si el número es igual a uno (`x == 1`):**
     ```python
     elif x == 1:
         print('Single')
     ```
     - Si el usuario ingresa uno, se imprime un mensaje informando que el número es uno.
   
   - **Caso contrario (default):**
     ```python
     else:
         print('More')
     ```
     - Si el número no cumple con ninguna de las condiciones anteriores, se imprime un mensaje informando que el número es mayor que uno.

## ¿Para qué sirven las estructuras condicionales?

Las estructuras `if`, `elif`, y `else` son fundamentales en la programación. Permiten que el programa ejecute diferentes bloques de código basado en ciertas condiciones. Esto es útil para:

- Validar entradas del usuario.
- tomar decisiones en juegos o aplicaciones interactivas.
- Procesar datos de acuerdo a reglas específicas.

## Mejores prácticas

- **Utiliza `elif` en lugar de múltiples `if`:** Esto hace el código más limpio y eficiente, ya que evita ejecutar múltiples condiciones si una anterior ha sido satisfecha.
  
- **Indentación correcta:** En Python, la indentación es crucial. Asegúrate de que todas las líneas dentro de un bloque condicional estén correctamente identadas (generalmente 4 espacios).

## Conclusión

Las estructuras condicionales son una herramienta poderosa en el arsenal del programador. El ejemplo que hemos explorado demuestra cómo Python puede manejar diferentes casos con simplicidad y claridad.
