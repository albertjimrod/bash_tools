# Git Tools

Herramientas para búsqueda y gestión de repositorios Git.

## Scripts disponibles

| Directorio | Descripción |
|------------|-------------|
| [buscar_git](./buscar_git/) | Busca repos Git en un directorio |
| [git-find-commit](./git-find-commit/) | Busca commit en múltiples repos |

## Requisitos

- Git instalado
- Bash 4.0+
