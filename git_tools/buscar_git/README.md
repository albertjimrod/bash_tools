# buscar_git

## Descripción

Directorio base donde buscar (por defecto, el directorio actual)
Buscar directorios que contengan una carpeta .git

## Uso

```bash
./buscar_git.sh [directorio]
```

## Requisitos

- Git instalado
- Bash 4.0+

## Código fuente

Ver `buscar_git.sh` para implementación completa.
