# yt_channel_to_corpus

## Descripción

yt_channel_to_corpus.sh
Extrae subtítulos de un canal de YouTube y genera un corpus Markdown

Uso:
./yt_channel_to_corpus.sh @NombreCanal

## Uso

```bash
./yt_channel_to_corpus.sh [argumentos]
```

## Requisitos

- yt-dlp (`pip install -U yt-dlp`)
- Bash 4.0+

## Código fuente

Ver `yt_channel_to_corpus.sh` para implementación completa.
