# make_corpus

## Descripción

make_corpus.sh
Crea un corpus Markdown a partir de archivos de texto (.txt, .md, etc.) con metadatos en formato:
# Título: ...
# URL: ...

## Uso

```bash
./make_corpus.sh [argumentos]
```

## Requisitos

- yt-dlp (`pip install -U yt-dlp`)
- Bash 4.0+

## Código fuente

Ver `make_corpus.sh` para implementación completa.
