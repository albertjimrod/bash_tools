# debug_video

## Descripción

debug_video.sh - Prueba de descarga de subtítulos para un vídeo específico

## Uso

```bash
./debug_video.sh [argumentos]
```

## Requisitos

- yt-dlp (`pip install -U yt-dlp`)
- Bash 4.0+

## Código fuente

Ver `debug_video.sh` para implementación completa.
