# Conda Tools

Herramientas para gestionar entornos Conda.

## Scripts disponibles

| Directorio | Descripción |
|------------|-------------|
| [check_conda_package](./check_conda_package/) | Busca un paquete en todos los entornos |
| [comparador_entornos](./comparador_entornos/) | Compara paquetes entre entornos |
| [comparador_entornos_new](./comparador_entornos_new/) | Versión mejorada del comparador |
| [conda_env_manager](./conda_env_manager/) | Gestor avanzado de entornos v2.0 |
| [conda_modulos_peso](./conda_modulos_peso/) | Muestra peso de módulos instalados |
| [demo](./demo/) | Demo del Conda Environment Manager |
| [export_conda_envs](./export_conda_envs/) | Exporta entornos a otra máquina |
| [export_conda_envs_de2c8d](./export_conda_envs_de2c8d/) | Versión alternativa de exportación |
| [import_conda_envs](./import_conda_envs/) | Importa entornos desde YAML |
| [install_manager](./install_manager/) | Instalador del Environment Manager |

## Uso

```bash
cd nombre_script/
./nombre_script.sh
```

## Requisitos

- Conda/Miniconda instalado
- Bash 4.0+
