# Conda Tools

Herramientas para gestionar entornos Conda.

## Scripts

| Script | Descripción | Uso |
|--------|-------------|-----|
| `conda_env_manager.sh` | Gestor avanzado de entornos Conda v2.0 | `./conda_env_manager.sh` |
| `check_conda_package.sh` | Busca un paquete en todos los entornos | `./check_conda_package.sh` |
| `comparador_entornos.sh` | Compara paquetes entre entornos | `./comparador_entornos.sh` |
| `comparador_entornos_new.sh` | Versión mejorada del comparador | `./comparador_entornos_new.sh` |
| `conda_modulos_peso.sh` | Muestra el peso de módulos instalados | `./conda_modulos_peso.sh` |
| `export_conda_envs.sh` | Exporta entornos a otra máquina | Editar IPs y ejecutar |
| `import_conda_envs.sh` | Importa entornos desde archivos YAML | `./import_conda_envs.sh` |
| `install_manager.sh` | Instalador del Conda Environment Manager | `./install_manager.sh` |
| `demo.sh` | Demo del Conda Environment Manager | `./demo.sh` |

## Requisitos

- Conda/Miniconda instalado
- Bash 4.0+
