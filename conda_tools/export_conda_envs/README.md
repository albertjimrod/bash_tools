# export_conda_envs

## Descripción

Configuración
Crear una carpeta temporal para almacenar los archivos YAML
Exportar todos los entornos de Conda excepto el base

## Uso

```bash
./export_conda_envs.sh
```

## Código fuente

Ver `export_conda_envs.sh` para implementación completa.
